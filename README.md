# My Cybersecurity & Software Engineering Journey

This repository is a private log of my path from computer engineering
student toward becoming a cybersecurity engineer. I'm documenting
everything here  the code I write, the labs I break, the notes I take,
and the projects I build — as I go.

I'm building in public for a few reasons: it keeps me accountable, it
forces me to actually finish things, and it leaves a trail others on the
same path can learn from.

## What you'll find here
- **Python** from fundamentals through OOP, plus small tools and scripts
- **C & low-level**  systems programming and the foundations that make
  security make sense
- **Networking & labs**  notes and write-ups from hands-on practice
- **Security write-ups**  documented solutions and lessons from CTFs and
  practice machines
- **Projects** everything I build along the way, rough edges and all

## Why
I come from a computer engineering background, which means I care about
understanding things from the hardware up not just running tools, but
knowing what's actually happening underneath. This repo tracks that
process honestly: the wins, the broken builds, and everything learned in
between.
